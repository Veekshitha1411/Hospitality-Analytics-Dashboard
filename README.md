# Hospitality-Analytics-Dashboard
A data analytics project developed using Power BI, SQL, Excel, and Tableau to analyze hotel revenue, occupancy, bookings, and cancellations across 7 properties.
## Tools Used
- Power BI
- SQL
- Excel


## Project Highlights
- Analyzed 134,590 hotel bookings
- Evaluated ₹1.71 Billion revenue
- Monitored 70.15% occupancy rate
- Identified 24.83% cancellation rate
